# Bedrud Frontend Platform

React 19 SPA. `apps/web/`. TanStack Router + TanStack Query 5 + TailwindCSS v4 + Bun.
Path alias: `#/*` тЖТ `./src/*`. Never `../src/*`.

---

## Entrypoints

### `src/router.tsx`

`getRouter()` тАФ TanStack Router instance. `scrollRestoration: true`, `defaultPreload: 'intent'`, `defaultPreloadStaleTime: 0`. Default not-found: `<ErrorPage variant="not-found" />`.

### `src/routes/__root.tsx` тАФ App shell

`QueryClientProvider` + `ErrorBoundary` wrapper (class component, keyed by `pathname+search` for auto-reset). `QueryClient` config: `retry: 1`, `staleTime: 30_000`. Inline theme script prevents dark-mode flash. Auth init: fire-and-forget `useAuthStore.getState().initialize()` on mount.

### `src/routeTree.gen.ts`

Auto-generated by TanStack Router. **Never edit.** 26 files, 6 layouts, 20 leaves.

## API Layer

### `src/lib/api.ts`

Central HTTP client. Methods: `api.get<T>(path, opts?)`, `api.post<T>`, `api.put<T>`, `api.delete<T>`.

- **Base URL:** `VITE_API_URL` env or empty (dev: Vite proxy `/api` тЖТ `localhost:8090`)
- **Auth:** `Authorization: Bearer <token>` when token exists. Fallback `X-CSRF-Token` from `<meta>` or cookie
- **Refresh:** 401 тЖТ `POST /api/auth/refresh` (singleton promise dedup) тЖТ retry once. Fail тЖТ redirect `/auth`
- **Credentials:** `include` on all requests
- **Error:** Non-OK throws `Error("{status}: {body text}")`
